CSS History Table
Year Event Notes
December 1996 CSS 1 specification becomes official Specification started with font properties, colors, text spacing, alignment, margins, borders, padding, positions, etc.
May 1998 CSS 2 specification published and recommended Additional positioning, media type concept, additional font/text properties added.
June 1999 CSS 3 initial drafts Backwards compatibility with CSS 2. Introduces modules to replace single specification.
2005 - 2011 CSS 2.1 recommendation and updates Fixed some errors in CSS 2.
August 2011 Bootstrap initial release Bootstrap is one of the largest HTML, CSS, and JS libraries used to date, offering developers modular CSS implementation methods. Bootstrap version 5 just released in May 2021.
Current CSS 3 "evolution" Individual module specifications "level" (develop) at different rates. For example, css3-multicol is in Working Draft status as of 02/2021, while mediaqueries-4 is in Candidate Recommendation status as of 06/2020.
Sources Wikipedia - CSS | Wikipedia - Bootstrap

Additionally, modern web browsers have contributed to the development of CSS, and the web in general, by adding support for new CSS features as they are developed and adopted by the W3C.
